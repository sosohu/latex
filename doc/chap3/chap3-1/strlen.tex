    \item{\textbf{问题}}: 求出字符串的长度.
    \item{\textbf{Care}} : \fbox{时间复杂度O(n), 空间复杂度O(1)}
    \\考虑输入指针不为空,输入参数使用const char
    \begin{lstlisting}
int strlen(const char *str){
	int len = 0;
	assert(str);  //判断不为NULL
	while(*str++ != '\0'){
		len++;
	}
	return len;
}
